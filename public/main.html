Draw with code!

Need help?

Shapes

circle(radius)

Draw a circle centered on the current position

  • radius: The circle size

ellipse(radius-x, radius-y)

Draw an ellipse centered on the current position

  • radius-x: The ellipse horizontal size
  • radius-y: The ellipse vertical size

square(size)

Draw a square centered on the current position

  • size: The size of the square

rectangle(width, height)

Draw a rectangle centered on the current position

  • width: The rectangle width
  • height: The rectangle height

Colors

color(color)

Change the color in use

  • color: The color to use

stroke([ size ], [ color ])

Change the line color and width in use

  • size: (number) The line width to use
  • color: A string with the color to use - E.g. 'red', 'blue'..

strokeColor(color)

The stroke color to use

  • color: A string with the color to use - E.g. 'red', 'blue'..

strokeWidth(size)

The stroke width to use

  • size: A number with the line width to use

Lines

line(x, [ y ])

Draw a line starting from the current position

  • x: Horizontal distance
  • y: Vertical distance

lineTo(x, y)

Draw a line to any position

  • x: The horizontal destination
  • y: The vertical destination

Position

move(x, [ y ])

Move the cursor starting from the current position

  • x: Horizontal distance
  • y: Vertical distance

moveTo(x, y)

Move the cursor to any position

  • x: The horizontal destination
  • y: The vertical destination

General

clear()

Clear the sketch

reset()

Reset styles
