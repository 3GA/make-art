## Setup

    git clone git@github.com:tancredi/draw.git
    cd draw
    npm install

## Build

Build the app before running it

    npm run build

## Run

    npm start

Open your browser at [http://localhost:9000](http://localhost:9000)

## Develop

Run the watch script when developing

    npm run watch
